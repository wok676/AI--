# UI/UX 设计规范

> 状态:**TBD(待定题)**。由 `ui-ux-designer` 在前端编码**之前**产出。模板见 `docs/prompts/templates/01b-ui-design.md`。

定题后本文件需包含:
1. 设计原则 + 一句话气质 + 2–3 个对标 App
2. 设计令牌(配色/字体/间距/圆角/阴影,**具体数值**可 copy 进 `theme.ts`)
3. 组件规范(按钮/输入/卡片/列表项/Tab/空态/骨架屏/Snackbar)
4. 信息架构与导航
5. 关键页面 ASCII 线框 + 三态
6. 交互动效、无障碍适配、图标品牌

> **任何与本规范不符的前端视觉都视为 bug。**
